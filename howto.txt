ClinicalTriage-Env — Setup and Testing Guide
=============================================

This guide provides step-by-step instructions to run, test, and verify the ClinicalTriage-Env reinforcement learning environment.

1. Local Setup
--------------
A. Environment Variables
   You need the following environment variables (required by the inference agent):
   - API_BASE_URL: Your OpenAI-compatible LLM provider URL (e.g., Hugging Face Inference API).
   - MODEL_NAME: The model you want to use (e.g., meta-llama/Llama-3.1-8B-Instruct).
   - HF_TOKEN: Your Hugging Face API token.

B. Installation
   1. Create a virtual environment:
      python -m venv venv
      .\venv\Scripts\activate  (Windows)
      source venv/bin/activate (Linux/Mac)
   2. Install dependencies:
      pip install -r requirements.txt
      pip install pytest  (for testing)

2. Running the Project
----------------------
Step 1: Start the OpenEnv Server
   uvicorn server:app --host 0.0.0.0 --port 7860

   Expected Output:
   INFO:     Started server process [PID]
   INFO:     Waiting for application startup.
   INFO:     Application startup complete.
   INFO:     Uvicorn running on http://0.0.0.0:7860

Step 2: Run Compliance Tests (In a new terminal)
   python tests/test_compliance.py

   Expected Output:
   - A list of passed tests marked with [PASS].
   - Final line: "All checks passed! Results: 33 passed, 0 failed" or similar.

Step 3: Run the Inference Agent (In a new terminal)
   Set your environment variables first (see section 1A), then:
   python inference.py

   Expected Output:
   - [START] task=single_patient_easy episode=1
   - [STEP] step=1 action={...} reward=0.XXXX
   - [END] task=single_patient_easy episode=1 total_reward=0.XXXX steps=1
   - ... (repeated for all tasks and episodes)
   - A final summary table showing "Avg Reward" per task.

3. Verification
---------------
The project is working properly if:
1. The server starts without errors on port 7860.
2. `tests/test_compliance.py` returns 0 failures.
3. `inference.py` completes all episodes and prints the [START]/[STEP]/[END] logs exactly as required.
4. Total rewards are between 0.0 and 1.0.

4. Running on Google Colab
--------------------------
To run this project on Google Colab, use the following steps in a Colab notebook:

Cell 1: Setup Environment
   !git clone <your-repo-url>  # Or upload your files manually
   %cd ClinicalTriage-Env
   !pip install -r requirements.txt pytest uvicorn fastapi pydantic openai httpx python-dotenv pyyaml

Cell 2: Start Server in Background
   import subprocess
   import time
   # Start uvicorn in the background
   server_proc = subprocess.Popen(["uvicorn", "server:app", "--host", "0.0.0.0", "--port", "7860"])
   time.sleep(5)  # Wait for server to boot

Cell 3: Run Inference Agent
   import os
   os.environ["API_BASE_URL"] = "https://router.huggingface.co/v1"  # Or your URL
   os.environ["MODEL_NAME"] = "meta-llama/Llama-3.1-8B-Instruct"
   os.environ["HF_TOKEN"] = "your_hf_token_here"
   
   !python inference.py

Cell 4: Run Tests
   !python tests/test_compliance.py

Cell 5: Stop Server (Optional)
   server_proc.terminate()
